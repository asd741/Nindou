# 忍豆工具箱

> 忍豆風雲4 線上工具箱 — 傷害計算器・公會戰福袋抽籤
>
> **線上使用：** https://asd741.github.io/Nindou/

---

## 功能介紹

### 傷害計算器
依照忍豆風雲4的傷害公式，精確計算每次攻擊的實際傷害：

- 基礎攻防（ATK / DEF）輸入
- 承受倍率（BAM）自動計算，附詳細步驟展開
- 傷害減免（DRM）自動計算，附詳細步驟展開
- 暴擊倍率（CM）一鍵設定
- 屬性相剋（elemental multiplier）選擇
- 攻擊側／防禦側所有參數皆可覆蓋
- 一鍵重置

### 公會戰福袋抽籤
公平、透明的公會戰福袋抽籤工具：

- 新增任意數量的籤（參與者名稱）
- 點擊籤條即可加入／移出轉盤
- 支援「全入」「全出」快速操作
- 賭場風格旋轉轉盤，動畫流暢
- 抽籤結果即時顯示，附歷史紀錄
- 支援自訂狀態標籤（例：已領取・未領取）並記憶在 localStorage

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建構正式版本（輸出至 docs/）
npm run build
```

建構完成後將 `docs/` 資料夾連同原始碼一起 push，GitHub Pages 即自動更新。
